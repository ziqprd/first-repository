# Flask blog

**Описание**

Блог где можно постить, лайкать, комментировать, и.т.д.

**Технологии**

 - Python 3
 - Flask
 - SQlite
 - HTML/CSS

**Функционал**

 - Регистрация и авторизация
 - Создание и редактирование постов
 - Удаление постов
 - Лайки
 - Добавление комментариев
 - Хранение данных в SQlite
